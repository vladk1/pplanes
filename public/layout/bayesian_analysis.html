BCBA provides functions to analyze a project under uncertain cost and benefits.

BCBA firstly simulate cases by using Monte Carlo Method. With a large amount of simulated cases, BCBA can do analyze base on these. In these case, BCBA computes the expected cost (ECost), expected benefit (EBenefit), expected net benefit (ENB), loss probability (LP) and 1%-value-at-risk (VaR) for each project. Based on these values, users can decide which project is better.
