• BCBA provides functions to analyze a project under uncertain cost and benefits.

• BCBA firstly simulate cases by using Monte Carlo Method. With a large amount of simulated cases, BCBA can do analyze base on these. In these case, BCBA computes the expected cost (ECost), expected benefit (EBenefit), expected net benefit (ENB), loss probability (LP) and 1%-value-at-risk (VaR) for each project. Based on these values, users can decide which project is better.

In probability theory, the expected value of a random variable is intuitively the long-run average value of repetitions of the experiment it represents

Expected cost (ECost) is the average value of cost of all generated cases.

Expected benefit (EBenefit) is the average value of benefit of all generated cases.

Expected net benefit (ENB) is the average value of net benefit of all generated cases. (Can be calculated by expected net benefit = expected benefit – expected cost)

Loss probability (LP) is the probability of that loss occurring.

Value-at-risk (VaR) is risk measure of the risk of lost on a specific case.
