Theory behind estimation

The situation, looks like this: you have two planes with different properties and you need to play one of them. Probably you have no idea on how difficult it is to build any of the planes or how would they perfom. Or maybe you can say "it couldn't be less than this and more than this". In fact, this is how you deal with uncertainty, you can use estimation intervals. But what can you do with this?

We use The Bayesian Cost Benefit Analysis inspired tool which computes Monte Carlo simulation over each interval of estimation given by you - building time and flight distance.

The intervals are treated as 95% confidence interval with normal distribution. Therefore, the actual value will be within the interval with 95% confidence. Some of the values would be outside the interval. When you do this, you need to have a 95% confidence in the interval estimated.

From the sampled data, the tool computes the expected value of each interval. This is represented by the mean value of the samples.

Then it calculates the Expected Net Benefit, Value at Risk and Loss Probability.
