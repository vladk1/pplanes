More on the decision-making tool:

• Monte-Carlo simulation is a broad class of computational algorithms that rely on repeated random sampling to obtain numerical results. The method simulates random input from a probability distribution over the domain. It is useful when problems are difficult or impossible to solve by other methods. However it only simulates, but not analyzes.

• The method requires to estimate values for both cost and benefit, including its minimum and maximum. Then the system can simulate cases as much as we want with a given type of distribution.

• Here, you estimate the values for minimum cost, maximum cost, minimum benefit, maximum benefit and regard both cost and benefit as normal distributions.
