R tool is a tool that specify for R language. R language is a programming language that is well used in statistical computing and graphics fields.

R tool supports self-made packages. In this case, we import a package called BCBA (Bayesian Cost-Benefit Analysis), which is written by Emmanuel Letier. This package provides specialized functions using Monte Carlo method to simulate with the estimate input. With the result, the system can create an analysis of project portfolios.
